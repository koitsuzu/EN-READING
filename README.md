# EN-READING: 你的 AI 專屬英文學習助手

這是一個讓你在網路上隨時隨地都能練習英文的強大工具！包含一個專業的學習網站，以及一個強大的瀏覽器外掛（擴充功能）。

## 🌟 兩大核心功能

### 1. 創新的英文學習網站
- **想要什麼文章，AI 寫給你**：你可以輸入感興趣的主題，讓 AI 為你量身打造適合你難度的文章。
- **邊讀邊學不求人**：文章中的單字只要點一下，立刻顯示中文翻譯與對應的英文定義。
- **讀完馬上測驗**：針對你讀的內容自動產生小考，並附上親切的中文解析，幫你檢查懂了多少。
- **成長進度看得到**：內建學習日曆，自動紀錄你讀過的文章量，讓你看到自己的進步。

### 2. 「收藏家」瀏覽器外掛 (滑到哪學到哪)
- **全世界都是你的教材**：在任何英文網頁、新聞中看到想研究的段落，點點右鍵就能直接送到學習網站分析。
- **貼心側邊欄助手**：不用離開目前的網頁，側邊欄就能幫你翻譯單字、朗讀全文。
- **數據自動同步**：你在外掛裡存下來的單字，主網站也會幫你收納好，學習紀錄完全不間斷。
- **簡單明瞭的統計**：側邊欄會同步顯示你的累積閱讀量，隨時隨地都能追蹤學習里程碑。

---

## 🛠️ 如何開始使用？

1. **安裝外掛**：將 `extension` 資料夾載入至 Chrome 瀏覽器的擴充功能管理頁面。
2. **設定金鑰**：進入設定頁面貼上你的 Google Gemini API Key。
3. **開始學習**：在網頁上點擊右鍵「抓取全文」或「傳送到側邊欄」，開始你的 AI 英文學習之旅！
